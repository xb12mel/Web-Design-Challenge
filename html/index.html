Summary: Latitude vs. "X"

The purpose of this project was to analyze how weather changes as you get closer to the equator. To accomplish this analysis, we first pulled data from the OpenWeatherMap API to assemble a dataset on over 500 cities.






Our objective was to build a series of scatter plots to showcase how ("X") changes with the latitude of the city.

  • Temperature (°F)
  • Humidity (%)
  • Cloudiness (%)
  • Wind Speed (MPH)

This site provided the source data and visualizaitons created as a part of the analysis, as well as explainations and descriptions of andy trends and correlations withnessed.
